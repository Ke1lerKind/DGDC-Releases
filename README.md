# DGDC - Drumgod Deathcount

Official release download page for DGDC.

DGDC is a local deathcount tool for Twitch streams, OBS overlays and Streamer.bot integration.

## Download

Go to the latest release and download the Windows installer.

## Included

- Windows setup installer
- German manual
- English manual

## Not included

The source code is not included in this repository.
